# Privacy Policy

[Japanese version](PRIVACY_POLICY.md)

Last updated: May 24, 2026

Twitch Comment Overlay ("this extension") is a Chrome extension that displays Twitch chat as an overlay on the watch page.

## Information Collected or Used

This extension reads the following information in the browser to provide its features:

- Chat message text on Twitch watch pages
- Twitch chat usernames
- Page content needed for overlay display, such as the Twitch chat area
- Settings such as display language, font size, scroll speed, opacity, row count, display area, and whether usernames are shown

## Purpose of Use

Chat messages and usernames are used only to display comments on the Twitch watch page.

Settings are used only to keep the same display preferences for future use.

## Storage

Extension settings are stored in Chrome local storage.

Chat messages and usernames are used temporarily for overlay display and display checks. This extension does not store that information on external servers.

## External Transmission and Third-Party Sharing

This extension does not send chat messages, usernames, settings, or page content to external servers.

User data is not sold, shared, or transferred.

## Limited Use of User Data

User data is used only for the single purpose of displaying Twitch chat as an overlay on the watch page.

This extension does not use user data for advertising, marketing, analytics, user tracking, or any purpose unrelated to the extension's stated feature.

The developer does not view, store, or process chat messages, usernames, settings, or page content obtained through this extension.

## Remote Code

This extension does not use remote code. All JavaScript, CSS, and HTML are included in the extension package, and no code is loaded or executed from external servers.

## Permissions

This extension uses Chrome extension permissions only as needed to display the comment overlay on Twitch watch pages and save user settings.
