# Twitch Comment Overlay

Twitch のチャットをブラウザ上にオーバーレイ表示する Chrome 拡張機能です。

## インストール

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このリポジトリのフォルダを選択する
5. 拡張機能の詳細画面にある「拡張機能のオプション」から設定画面を開ける

## 使い方

1. `https://www.twitch.tv/` の配信ページを開く
2. チャットが流れると、ページ上にコメントが右から左へ表示される
3. 拡張機能アイコンのポップアップから表示、文字サイズ、速度、透明度、行数、表示範囲、ユーザー名表示を調整する

## 表示確認

1. `chrome://extensions/` で拡張機能の再読み込みボタンを押す
2. Twitch の配信ページを再読み込みする
3. 拡張機能アイコンを押して「表示テスト」を押す
4. 画面右端から `Overlay test message` が流れれば、オーバーレイ表示は動作している
5. ポップアップ下部の「状態」で、チャット欄の監視状況と検出数を確認する

「表示テスト失敗」と出る場合は、Twitch の配信ページをアクティブにした状態で拡張機能アイコンを開いてください。
権限やスクリプトを変更した後は、`chrome://extensions/` で拡張機能の再読み込みが必要です。

## 設定画面

Chrome の拡張機能詳細画面から開く設定画面でも、ポップアップと同じ設定項目を変更できます。
変更した値は `chrome.storage.local` に保存され、ブラウザ内に保持されます。

## パッケージ作成

`dist-package` を作成するスクリプトを用意しています。

- Windows (cmd):
  - `build-package.bat`
- Linux/macOS (bash):
  - `./build-package.sh`

どちらも `dist-package` を再作成し、配布に必要なファイル一式をコピーします。

## 動作確認メモ

### 成功状態

- Twitch の配信ページをアクティブにした状態で拡張機能アイコンを開く
- 「表示テスト」を押す
- 画面右端から `Overlay test message` が流れる
- ポップアップの「状態」が `未確認` のままではなく、監視状態や検出情報に更新される

この状態を、オーバーレイ注入と表示処理が正常に動いている基準にする。

### 失敗状態と原因

- 失敗内容: 「表示テスト」を押しても何も反応せず、「状態」も `未確認` のまま
- 主な原因: Twitch タブ側に `content.js` が注入されておらず、ポップアップからのテスト要求が届いていなかった
- 対応: 「表示テスト」押下時に、アクティブな Twitch タブへ `settings.js` と `content.js` を明示注入し、`chrome.tabs.sendMessage` で直接テスト表示を要求するようにした
- 注意: `manifest.json` の権限変更後は、`chrome://extensions/` で拡張機能の再読み込みが必要

## 注意

Twitch の画面構造が変わると、コメント取得用のセレクタ調整が必要になる場合があります。

## 作業ルール（実装時）

`apply_patch` で「削除」が反映されないことがあるため、以下をルール化する。

1. 削除パッチ適用直後に必ず検証する  
   - `rg` / `sed` / `tail` / `wc -l` で対象行が消えているか確認する
2. 消えていない場合は、より広いコンテキストで再度 `apply_patch` する  
   - 前後行を多めに含めた hunk にする
3. それでも不発の場合は、`apply_patch` に固執しない  
   - `head` / `sed` / `awk` で一時ファイルを作成し、`mv` で置き換える
4. 置き換え後は必ず構文チェックとテストを実行する  
   - 例: `node --check <file>`、`npm run check`、`npm test`
