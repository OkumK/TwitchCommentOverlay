# Twitch Comment Overlay

Twitch のチャットをブラウザ上にオーバーレイ表示する Chrome 拡張機能です。

## インストール

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このリポジトリのフォルダを選択する
5. 拡張機能の詳細画面にある「拡張機能のオプション」から設定画面を開く

## 使い方

1. `https://www.twitch.tv/` の配信ページを開く
2. チャットが流れると、ページ上にコメントが右から左へ表示される
3. 拡張機能アイコンのポップアップから表示、言語、文字サイズ、速度、透明度、行数、表示範囲、ユーザー名表示を調整する

## 設定画面

Chrome の拡張機能詳細画面から開く設定画面でも、ポップアップと同じ設定項目を変更できます。
変更した値は `chrome.storage.local` に保存され、ブラウザ内に保持されます。
言語設定で、ポップアップと設定画面を日本語または英語に切り替えられます。

## 注意

Twitch の画面構造が変わると、コメント取得用のセレクタ調整が必要になる場合があります。
